# Proyecto de Cliente-Servidor en C

Este proyecto contiene dos archivos principales: `cliente.c` y `servidor.c`.

## Descripción

Este proyecto implementa una comunicación simple entre cliente y servidor utilizando el lenguaje de programación C.

## Archivos

El proyecto consta de dos archivos principales:

1. `cliente.c`: Este archivo contiene el código para el cliente que se conecta al servidor.
2. `servidor.c`: Este archivo contiene el código para el servidor al que se conecta el cliente.

## Cómo ejecutar

Para compilar los archivos, puedes usar el compilador gcc de la siguiente manera:

```bash
gcc servidor.c -o servidor
gcc cliente.c -o cliente
```

Para ejecutar los archivos compilados, puedes hacerlo de la siguiente manera:

```bash
./servidor
./cliente
```

Posteriormente, ejecute los comandos que desea y el servidor se encargará de retornar el resultado.
Al momento que desea salir, ingrese exit.
